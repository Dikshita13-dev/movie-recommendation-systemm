# Movie Recommendation System

## Project Overview
A recommendation system built using Python, Pandas, and Scikit-Learn. Includes:
- Content-based filtering (based on genres)
- Collaborative filtering (based on user ratings)
- EDA and visualizations

## Dataset
MovieLens dataset provided by GroupLens Research, University of Minnesota: [MovieLens](https://grouplens.org/datasets/movielens/)

## Folder Structure
- `notebooks/` : All Jupyter notebooks
- `data/` : Dataset CSV files (optional if large)
- `screenshots/` : Example plots from EDA

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-Learn, Surprise

## Features
- EDA: genre trends, ratings distribution, top movies
- Content-based recommender: recommends movies based on genres
- Collaborative filtering: recommends movies based on user ratings
